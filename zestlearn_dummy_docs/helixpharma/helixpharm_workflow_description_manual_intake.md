# HelixPharm Solutions AG — Workflow Description: Manual Document Intake

## Current State
The document intake process is mostly manual and includes opening incoming files, checking basic completeness, assigning categories, and forwarding them to the right reviewer.

## Current Tools
- Shared inbox
- Shared folders
- Excel approval tracker
- Word templates

## Frictions
- Intake relies on coordinator attention
- Similar files are processed repeatedly from scratch
- Important metadata is sometimes missing
- Reviewers receive files without a clear summary

## Business Problem
Too much time is spent on repetitive document handling before value-added review even begins.

## Potential Pilot
A triage assistant that summarizes documents, proposes categories, and flags missing sections before human review.
