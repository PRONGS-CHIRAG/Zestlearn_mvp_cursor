# Rheonix Life Sciences GmbH — Workflow Description: Status Tracking and Follow-Up

## Current State
Status tracking relies on spreadsheets, meeting notes, and email follow-ups. There is no single continuously updated project context layer.

## Current Steps
1. Teams maintain local updates
2. PMO requests updates manually
3. Notes are consolidated after meetings
4. Action items are sent by email
5. Follow-up happens through manual reminders

## Frictions
- Latest status is not always obvious
- Action items can be lost
- Teams need to ask repeatedly for context
- Coordination effort grows with project complexity

## Potential Pilot
An assistant that summarizes project updates, tracks action items, and provides a simple internal Q&A layer for current status.
